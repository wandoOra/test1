# wCashless Mobile application

## 1. Clone the project

`git clone gogs@forge.alphacogs.com:covernal/wCashless.git`

`cd wCashless.git`

`npm install`

## 2. Build up the native projects

`expo prebuild --clean --npm`

## 3. Run the app

### iOS

`expo run:ios`

### Android

`expo run:android`


## 4. Build native project

### iOS

`expo build:ios`

### Android

`expo build:android`
